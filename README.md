# Video Downloader UI

## Архітектура
- Десктопна оболонка побудована на `pywebview`: `main_webview.py` відкриває інтерфейс з HTML/JS/CSS у теці `web` (Tailwind CSS для стилів).
- Фонові завантаження та конвертація виконуються через `yt_downloader/worker.py`, `yt_downloader/backend.py` та утиліти з `yt_downloader/utils.py`.
- Усі необхідні ресурси (іконки, статичні файли `web`) пакуються разом із застосунком під час збірки.

## Як запустити на Windows
1. Завантажте останній реліз з розділу **Releases** на GitHub (`yt-downloader-*-windows.zip`).
2. Розпакуйте архів у будь-яку теку на вашому комп'ютері.
3. Запустіть `yt-downloader.exe` з розпакованої теки — утиліти `yt-dlp`, `ffmpeg`, `ffprobe` входять у комплект.
4. Під час першого запуску оберіть теку для збереження відео; шлях зберігається у `settings.json` поруч із виконуваним файлом.
5. Щоб отримати нову версію, повторно завантажте й розпакуйте актуальний реліз, замінивши попередню теку програми.
