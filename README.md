# YouTube Downloader UI

## Як запустити на Windows
1. Завантажте останній реліз з розділу **Releases** на GitHub (`yt-downloader-*-windows.zip`).
2. Розпакуйте архів у будь-яку теку на вашому комп'ютері.
3. Запустіть `yt-downloader.exe` з розпакованої теки — усі потрібні утиліти (`yt-dlp`, `ffmpeg`, `ffprobe`) вже входять у комплект.
4. Під час першого запуску оберіть теку для збереження відео; шлях зберігається у `settings.json` поруч із виконуваним файлом.
5. Щоб отримати нову версію, повторно завантажте й розпакуйте актуальний реліз, замінивши попередню теку програми.
